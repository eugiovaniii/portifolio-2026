# Portfólio Web - Giovani Pereira Macedo

Portfólio pessoal com foco em apresentação profissional, projetos práticos e evolução técnica.

## O que foi reestruturado

- Página principal organizada em seções semânticas (hero, sobre, experiência, projetos, skills e contato).
- Visual atualizado com CSS mais consistente, responsivo e sem dependência de Bootstrap/jQuery na página inicial.
- Conteúdo profissional revisado para destacar experiência real com suporte, testes e operação.
- Projeto `Loja-Virtual` transformado em demo funcional de catálogo + carrinho com `localStorage`.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Estrutura do projeto

```text
novo-portifolio-main/
├─ index.html
├─ style.css
├─ script.js
├─ assets/
└─ projetos/
   ├─ Loja-Virtual/
   │  ├─ index.html
   │  ├─ style.css
   │  └─ script.js
   ├─ ger-tarefas/
   ├─ meu-blog/
   └─ pagina-viagem/
```

## Como executar localmente

1. Baixe/clonar o repositório.
2. Abra `index.html` no navegador.
3. Para testar projeto específico, abra o `index.html` dentro da pasta do projeto.
