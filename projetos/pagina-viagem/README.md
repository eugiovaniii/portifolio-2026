# Página de Viagem

Landing page de agência de viagens com simulador de orçamento.

## Visão geral

Projeto front-end para demonstrar construção de página comercial completa: hero com slider, catálogo de pacotes, seção de destinos, tabela de serviços e formulário de estimativa.

## Funcionalidades

- Carrossel manual/automático com imagens de destaque.
- Seção de pacotes com proposta de valor.
- Grid de destinos com imagens locais do projeto.
- Tabela de serviços e taxas base.
- Formulário de orçamento com:
  - destino
  - quantidade de pessoas
  - quantidade de dias
  - datas de ida/volta
  - serviços adicionais
- Cálculo dinâmico da estimativa com breakdown dos custos.

## Estrutura

- `index.html`: estrutura da landing page e formulário.
- `style.css`: identidade visual, layout responsivo e componentes.
- `script.js`: lógica do slider e cálculo do orçamento.
- `src/img`: imagens utilizadas no projeto.

## Como executar

1. Abra `index.html` no navegador.
2. Navegue pelas seções e teste o slider.
3. Preencha o formulário de orçamento para ver a estimativa.
